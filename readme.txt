
A low level Python3 interface to access pkcs11 modules.

CFFI is used to access the C modules.

Related pages
=============

 * Definition of the standard : https://www.oasis-open.org/committees/documents.php?wg_abbrev=pkcs11

 * Readable documentation : https://nlnetlabs.nl/downloads/publications/hsm/hsm_node9.html

 * CFFI documentation : https://cffi.readthedocs.org/en/latest/index.html

 * A SWIG based interface : https://bitbucket.org/gambit4id/pykcs11/overview
